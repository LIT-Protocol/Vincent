---
title: 'Complete Uninstall'
api: 'POST /user/{appId}/complete-uninstall'
---

Complete the permission revocation by submitting the user's signature.

## Request

<ParamField path="appId" type="integer" required>
  The unique identifier of the app
</ParamField>

<ParamField body="typedDataSignature" type="string" required>
  The user's signature of the typed data from the uninstall-app response
</ParamField>

<ParamField body="uninstallDataToSign" type="object" required>
  The typed data object returned from the uninstall-app endpoint
</ParamField>

## Response

<ResponseField name="transactionHash" type="string">
  The transaction hash of the on-chain uninstall
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.heyvincent.ai/user/123/complete-uninstall \
  -H "Content-Type: application/json" \
  -d '{
    "typedDataSignature": "0xSignature...",
    "uninstallDataToSign": { "...": "..." }
  }'
```

```typescript TypeScript
// Step 1: Initiate uninstall
const uninstallResponse = await fetch(`https://api.heyvincent.ai/user/${appId}/uninstall-app`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    userControllerAddress: userWallet.address,
    appVersion: 1
  })
});
const uninstallData = await uninstallResponse.json();

// Step 2: Sign the typed data with the user's wallet
const { typedData } = uninstallData.uninstallDataToSign;
const { EIP712Domain, ...types } = typedData.types;
const signature = await userWallet._signTypedData(
  typedData.domain,
  types,
  typedData.message
);

// Step 3: Complete the uninstall
const response = await fetch(`https://api.heyvincent.ai/user/${appId}/complete-uninstall`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    typedDataSignature: signature,
    uninstallDataToSign: uninstallData.uninstallDataToSign
  })
});

const result = await response.json();
console.log(`Uninstall tx: ${result.transactionHash}`);
```
