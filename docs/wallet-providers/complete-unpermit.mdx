---
title: 'Complete Unpermit'
api: 'POST /user/{appId}/complete-unpermit'
---

Complete the permission revocation by submitting the user's signature.

## Request

<ParamField path="appId" type="integer" required>
  The unique identifier of the app
</ParamField>

<ParamField body="typedDataSignature" type="string" required>
  The user's signature of the typed data from the unpermit-app response
</ParamField>

<ParamField body="unpermitDataToSign" type="object" required>
  The typed data object returned from the unpermit-app endpoint
</ParamField>

## Response

<ResponseField name="transactionHash" type="string">
  The transaction hash of the on-chain unpermit
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.heyvincent.ai/user/123/complete-unpermit \
  -H "Content-Type: application/json" \
  -d '{
    "typedDataSignature": "0xSignature...",
    "unpermitDataToSign": { "...": "..." }
  }'
```

```typescript TypeScript
// Step 1: Initiate unpermit
const unpermitResponse = await fetch(`https://api.heyvincent.ai/user/${appId}/unpermit-app`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    userControllerAddress: userWallet.address,
    appVersion: 1
  })
});
const unpermitData = await unpermitResponse.json();

// Step 2: Sign the typed data with the user's wallet
const { typedData } = unpermitData.unpermitDataToSign;
const { EIP712Domain, ...types } = typedData.types;
const signature = await userWallet._signTypedData(
  typedData.domain,
  types,
  typedData.message
);

// Step 3: Complete the unpermit
const response = await fetch(`https://api.heyvincent.ai/user/${appId}/complete-unpermit`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    typedDataSignature: signature,
    unpermitDataToSign: unpermitData.unpermitDataToSign
  })
});

const result = await response.json();
console.log(`Unpermit tx: ${result.transactionHash}`);
```
